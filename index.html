HTML Table Cheatsheet

This is a cheatsheet for the tags and attributes for making tables in HTML

Table Tags

HTML Tag Tag Name Description
<table> Table Creates a table.
<tr> Table Row Adds a row to the table.
<td> Table Data Adds data to the table row going from left to right cells.
<th> Table Heading Clarifying the table heads of data.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tfoot> Table Footer Creates a table footer row, perfect for having sums and totals.

Table Attributes

Attribute Name Description
<colspan> Column Span Defines how many columns a td element should span. Similar to merging cells vertically.
<rowspan> Row Span Defines how many rows a td element should span. Similar to merging cells horizontally.
